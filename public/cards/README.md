# Card art drop folder

Drop one image per card here, named `<card id>.png` — it appears
automatically on board tokens, hand cards, and the mulligan screen.
No file = the flat colored token (nothing breaks).

Export from Canva: Share → Download → PNG, one page/card at a time.
Export the **artwork only** (no stat badges or text baked in) — the game
draws name, damage, shields, HP, and speed live on top, with a dark
gradient at the bottom for readability. Portrait crop, ~300×360 px or
larger; the top-center of the image stays most visible.
